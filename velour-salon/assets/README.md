# Assets Folder

Place your salon images and icons here.

## images/
Add your real salon photos here and replace the emoji placeholders in HTML with:
```html
<img src="../assets/images/your-photo.jpg" alt="Description" loading="lazy" />
```

Recommended image sizes:
- Hero background: 1920×1080px
- Service cards: 600×400px
- Team photos: 400×500px
- Gallery: 800×800px (square)

## icons/
Add any custom SVG icons here if needed.
